# Task-Manager-App

Task Manager App with neumorphism UI built using Node.js, Express.js, and MongoDB. With the feature to edit and delete a task.

# Visit Me!

[Task Manager App](https://taskmanager-api-v1.herokuapp.com/)

## Features

- Add a new task
- Edit an existing task
- Mark as completed
- Delete a task
